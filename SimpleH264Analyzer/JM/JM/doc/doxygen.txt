Doxygen usage information
-------------------------

Doxygen can be downloaded from http://www.doxygen.org

 Note: for generating graphs you will also need "dot" which is part of 
       the ATT Graphviz tools available at
       http://www.research.att.com/sw/tools/graphviz


For generation of the TML reference software documentation run

doxygen lencod.dox
doxygen ldecod.dox

in this directory.

The documentation will be genrated into the directories lenc/ and ldec/
